# Function: range()

```ts
function range(start, end?): number[]
```

Returns a list of integers from *start* to *end*.
(similar to python's range)

If only 1 value is passed, uses [0, start[.<br>
If 2 values are passed, uses [start, end[.

## Parameters

• **start**: `number`

• **end?**: `number`

## Returns

`number`[]
